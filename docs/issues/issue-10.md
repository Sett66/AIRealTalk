## 要构建什么（端到端纵向切片）

对全链路进行 **稳定性与体验打磨**，使项目达到 SPEC 7.4 **MVP 完成定义**。包括：WebSocket 自动重连、录音权限引导、处理超时与重试、Prompt 调优、本地启动文档完善。

## 验收标准

### MVP Done（必须全部满足）

- [ ] Android 真机 3 场景之一完成 ≥ 5 轮英语语音对话
- [ ] 对话中出现过轻提示，且 **不阻断** TTS
- [ ] 课后报告含发音均分 + ≥ 3 条表达纠错 + 语速/轮次统计
- [ ] 历史页 ≥ 3 条记录 + 分数趋势图
- [ ] `grep` mobile 目录无 API Key 泄露

### 稳定性项

- [ ] WS 断线自动重连（指数退避，最多 5 次）
- [ ] 录音权限被拒时显示引导页
- [ ] ASR 超时 15s、LLM 超时 20s 可重试
- [ ] README 含完整本地启动步骤
- [ ] （可选）iOS 编译冒烟，非必须
